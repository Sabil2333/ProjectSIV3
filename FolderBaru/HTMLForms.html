HTML Forms

Apa itu HTML Forms?

HTML Forms adalah elemen dalam HTML yang digunakan untuk mengumpulkan data dari pengguna dan mengirimkannya ke server untuk diproses. Formulir sering digunakan untuk berbagai keperluan, seperti login, pendaftaran, pencarian, atau pengisian informasi.

Struktur Dasar HTML Forms:
Formulir dalam HTML dibuat menggunakan elemen <form>. Elemen ini biasanya berisi berbagai jenis input, tombol, dan elemen lainnya yang memungkinkan pengguna untuk memasukkan informasi.


Contoh Coding Forms :

<form>
  <label for="name">Nama:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Kirim</button>
</form>

Output Yang dihasilkan adalah sebagai berikut :


Adapun Penjelasan Dari Code Diatas:

<form>: Elemen utama untuk mengirim data ke server.

action="tujuan.php": URL tujuan pengiriman data.

method="POST": Metode pengiriman data secara aman.

<label>: Memberikan keterangan untuk input, terhubung melalui atribut for.

<input>: Elemen untuk menerima input teks dari pengguna.

id="name": Identitas unik.

name="name": Nama variabel untuk pengiriman data.

<button>: Tombol untuk mengirimkan formulir.

HTML Forms Attribute

Apa itu Attributes Pada Forms?

HTML Form Attributes adalah properti tambahan yang digunakan untuk mengontrol cara formulir bekerja dan bagaimana data dikirimkan. Atribut ini ditambahkan ke elemen <form> untuk menentukan perilaku formulir, seperti metode pengiriman, URL tujuan, validasi, dan sebagainya.


Daftar List Atrributes Pada Forms

Atribut Keterangan
Action Attribute Menentukan ke mana akan mengirim data formulir setelah dikirimkan.
Target Menentukan tempat untuk menampilkan respons setelah pengiriman.
Method Menentukan metode HTTP yang digunakan untuk mengirimkan formulir.
autocomplete Mengaktifkan saran pengisian otomatis browser untuk kolom formulir.
novalidate Menonaktifkan validasi formulir sisi klien saat ada.
Rel Menentukan hubungan antara dokumen saat ini dan dokumen yang ditautkan.
accept-charset Menentukan pengkodean karakter yang digunakan untuk pengiriman formulir.
Enctype Menentukan bagaimana data formulir harus dikodekan untuk pengiriman.

HTML Input Type

Apa itu Input Type

Input Type adalah atribut pada elemen <input> dalam HTML yang digunakan untuk menentukan jenis data atau masukan yang akan diterima dari pengguna. Atribut ini memungkinkan pengembang untuk membuat berbagai jenis kolom input, seperti teks, angka, email, kata sandi, tanggal, dan lainnya.
