Sintaks Dasar C++


Apa itu Sintaks C++?

Sintaks dalam C++ adalah aturan atau tata cara penulisan kode yang harus diikuti agar program dapat dikompilasi dan dijalankan dengan benar. Setiap bahasa pemrograman memiliki aturan sintaksnya sendiri, dan memahami sintaks C++ adalah langkah pertama untuk menjadi seorang programmer yang andal.


Contoh Program Sederhana

Berikut adalah contoh program C++ sederhana yang mencetak "Halo, Dunia!" ke layar:

Contoh Program
#include <iostream>
using namespace std;

int main() {
    cout << "Halo, Dunia!" << endl;
    return 0;
}
Coba Program »

Penjelasan Sintaks

  • #include <iostream>: Menyertakan pustaka standar untuk input dan output.
  • using namespace std;: Membuat elemen dalam ruang nama std lebih mudah diakses.
  • int main(): Fungsi utama yang menjadi titik awal eksekusi program.
  • cout: Digunakan untuk mencetak teks ke layar.
  • return 0;: Menandakan bahwa program selesai dijalankan dengan sukses.

Aturan Sintaks yang Perlu Diketahui

  • Setiap perintah diakhiri dengan tanda titik koma (;).
  • Penulisan kode bersifat case-sensitive (huruf besar dan kecil dibedakan).
  • Fungsi main() adalah fungsi wajib dalam setiap program C++.
  • Gunakan indentasi untuk membuat kode lebih mudah dibaca.

Pernyataan dalam C++


Apa itu Pernyataan dalam C++?

Pernyataan dalam C++ adalah instruksi yang memberitahu komputer untuk melakukan suatu aksi. Setiap pernyataan diakhiri dengan titik koma (;) untuk menunjukkan akhir dari instruksi tersebut.


Contoh Pernyataan

Berikut adalah beberapa contoh pernyataan dalam C++:

Contoh Kode Pernyataan

int x = 10;              // Deklarasi variabel dan inisialisasi
x = x + 5;              // Operasi ekspresi
cout << x << endl;      // Pernyataan output
Coba Kode »

Jenis Pernyataan dalam C++

  • Pernyataan Deklarasi: Digunakan untuk mendeklarasikan variabel atau objek, contohnya int x;.
  • Pernyataan Ekspresi: Melakukan operasi pada nilai, contohnya x = x + 5;.
  • Pernyataan Kondisional: Mengontrol alur program berdasarkan kondisi, contohnya if atau switch.
  • Pernyataan Iterasi: Mengulang blok kode dengan perulangan, contohnya for, while, atau do-while.
  • Pernyataan Jump: Mengontrol alur program dengan melompat, contohnya break, continue, atau goto.

Struktur Dasar Pernyataan

Pernyataan dalam C++ selalu ditulis dengan struktur berikut: 

jenis_data nama_variabel = nilai_awal;   // Deklarasi variabel
aksi;                                  // Pernyataan ekspresi atau aksi

Tips Menulis Pernyataan dalam C++

  • Pastikan setiap pernyataan diakhiri dengan tanda titik koma (;).
  • Gunakan komentar untuk mendokumentasikan tujuan pernyataan.
  • Gunakan indentasi untuk meningkatkan keterbacaan kode.
