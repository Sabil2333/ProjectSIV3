Selamat Datang di Tutorial CSS


1. APA ITU CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan format elemen pada halaman web. Dengan CSS, kita dapat mengubah warna, tata letak, font, dan elemen visual lainnya pada sebuah website. CSS bekerja bersama HTML untuk memberikan desain yang menarik dan fungsional.



2. Fungsi CSS

CSS digunakan untuk:

  • Mengatur Tata Letak: Mengatur posisi elemen seperti header, sidebar, footer, dll.
  • Memberikan Gaya Visual: Mengubah warna, jenis font, ukuran teks, dan elemen grafis lainnya.
  • Meningkatkan Konsistensi: Dengan satu file CSS, tampilan banyak halaman dapat dikelola dengan mudah.
  • Memisahkan Konten dari Desain: Memudahkan pengelolaan kode karena HTML hanya berisi konten, sedangkan desain dikelola dengan CSS.


3. Cara Penulisan CSS

Ada tiga cara untuk menambahkan CSS ke dalam proyek web:

  1. Inline CSS
    • Ditulis langsung dalam atribut elemen HTML.
  2. Internal CSS

    • dalam tag < style > di bagian < head > dokumen HTML.

  3. External CSS
    • Disimpan dalam file terpisah dengan ekstensi .css.


4. Struktur Dasar CSS

Contoh penulisan struktur di CSS

  • Selector: Elemen HTML yang ingin diberikan gaya (contoh: p, h1, div).
  • Property: Properti yang ingin diatur (contoh: color, font-size).
  • Value: Nilai dari properti tersebut (contoh: blue, 16px).


5. Jenis Selector pada CSS

  • Universal Selector (*): Menerapkan gaya ke semua elemen
  • Type Selector: Menerapkan gaya ke elemen tertentu
  • Class Selector (.): Menerapkan gaya ke elemen dengan atribut class
  • ID Selector (#): Menerapkan gaya ke elemen dengan atribut id.


6. Properti CSS yang Sering Digunakan

  1. Teks
    • color: Mengatur warna teks.
    • font-size: Mengatur ukuran font.
    • text-align: Mengatur perataan teks (left, center, right, justify)
  2. Tata Letak
    • margin: Memberikan jarak di luar elemen.
    • padding: Memberikan jarak di dalam elemen.
    • border: Menambahkan garis tepi pada elemen.
  3. Warna dan Latar Belakang
    • background-color: Memberikan warna latar belakang.
    • background-image: Memberikan gambar sebagai latar belakang.
