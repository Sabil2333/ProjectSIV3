Cetak Teks dalam C++


Apa itu Cetak Teks dalam C++?

Untuk mencetak teks di C++, kita menggunakan objek cout yang merupakan bagian dari pustaka standar <iostream>. Objek ini bekerja dengan operator sisipan << untuk mengirim output ke layar. Pastikan untuk mengelilingi teks yang ingin dicetak dengan tanda kutip ganda (").


Contoh Program Sederhana

Berikut adalah contoh program C++ sederhana yang mencetak "Halo, Dunia!" ke layar:

Contoh Program
#include <iostream>
using namespace std;

int main() {
    cout << "Halo, Dunia!";
    return 0;
}
Coba Program »

Mencetak Lebih dari Satu Objek

Anda dapat mencetak beberapa teks atau objek dengan menggunakan beberapa pernyataan cout atau menggabungkannya dalam satu pernyataan: 

#include <iostream>
using namespace std;

int main() {
    cout << "Halo, Dunia!";
    cout << " Saya sedang belajar C++.";
    return 0;
}

Output Program

Program di atas akan menghasilkan output berikut (tanpa baris baru):

Halo, Dunia! Saya sedang belajar C++.

Cetak Angka dalam C++


Apa itu Cetak Angka dalam C++?

Untuk mencetak angka di C++, kita menggunakan objek cout dari pustaka standar <iostream>. Berbeda dengan teks, angka tidak perlu dikelilingi oleh tanda kutip. Kita dapat mencetak angka secara langsung atau melalui variabel yang menyimpan nilai numerik.


Contoh Program Sederhana

Berikut adalah contoh program C++ sederhana yang mencetak angka ke layar:

Contoh Program
  #include <iostream>
  using namespace std;
  
  int main() {
      cout << 3;
      return 0;
  }
Coba Program »

Menggunakan Variabel

Anda juga dapat mencetak angka yang disimpan dalam variabel: 

  #include <iostream>
  using namespace std;
  
  int main() {
      int myNum = 15;
      cout << myNum;
      return 0;
  }

Menggabungkan Teks dan Angka

Teks dan angka dapat digabungkan dalam satu pernyataan cout menggunakan operator <<. 

  #include <iostream>
  using namespace std;
  
  int main() {
      int myNum = 15;
      cout << "Nilai saya adalah: " << myNum;
      return 0;
  }

Menghitung dan Mencetak Hasil

Anda juga dapat mencetak hasil perhitungan langsung dalam pernyataan cout: 

  #include <iostream>
  using namespace std;
  
  int main() {
      cout << 3 + 3;
      return 0;
  }

Output Program

Program di atas akan menghasilkan output berikut:

6

Baris Baru dalam C++


Apa itu Baris Baru dalam C++?

Dalam C++, kita dapat membuat baris baru dalam output menggunakan karakter newline \n atau manipulasi endl. Keduanya memiliki fungsi yang sama, yaitu memindahkan kursor ke baris berikutnya, namun dengan cara kerja yang berbeda.


Karakter Newline \n

Karakter \n digunakan untuk menyisipkan baris baru dalam output tanpa melakukan flush pada buffer. Penggunaan ini efisien untuk output yang sederhana. 

#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!\n";
    cout << "I am learning C++";
    return 0;
}

Manipulator endl

Manipulator endl tidak hanya menambahkan baris baru tetapi juga melakukan flush pada buffer output. Hal ini berguna dalam situasi tertentu seperti penulisan ke file atau perangkat output yang membutuhkan sinkronisasi data. 

#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    cout << "I am learning C++" << endl;
    return 0;
}

Perbandingan \n dan endl

Keduanya menghasilkan efek visual yang sama, namun dengan perbedaan berikut:

  • \n: Tidak melakukan flush buffer sehingga lebih cepat.
  • endl: Melakukan flush buffer, berguna untuk sinkronisasi tetapi kurang efisien jika digunakan berlebihan.
