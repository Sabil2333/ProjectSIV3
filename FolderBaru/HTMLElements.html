HTML Color

Seputar Mengenai Color HTML

HTML Color adalah fitur dalam HTML yang digunakan untuk menentukan warna elemen pada halaman web. Warna dapat diterapkan pada teks, latar belakang, border, atau elemen lainnya menggunakan atribut HTML atau CSS.

Berikut adalah contoh dari beberapa warna yang ada di HTML :

HTML Colors

Tomato
Orange
DodgerBlue
MediumSeaGreen
Gray
SlateBlue
Violet
LightGray

HTML Mendukung 140 standard color names.


Adapun Codingan dari beberapa Warna di atas :

Contoh Sederhana
<h1 style="background-color:Tomato;">Tomato</h1>
<h1 style="background-color:Orange;">Orange</h1>
<h1 style="background-color:DodgerBlue;">DodgerBlue</h1>
<h1 style="background-color:MediumSeaGreen;">MediumSeaGreen</h1>
<h1 style="background-color:Gray;">Gray</h1>
<h1 style="background-color:SlateBlue;">SlateBlue</h1>
<h1 style="background-color:Violet;">Violet</h1>
<h1 style="background-color:LightGray;">LightGray</h1>
Try it Yourself »

Penjelasan Contoh Warna HTML: Dalam contoh ini, kita membuat HTML untuk menyusun konten menggunakan elemen seperti heading, div, link, dan paragraf.

Nilai Warna: Nilai warna dalam HTML digunakan untuk menentukan warna elemen. Nilai ini dapat ditentukan menggunakan berbagai format seperti hexadecimal, RGB, RGBA, HSL, HSLA, nama warna, dan kata kunci warna sistem.

Nilai Warna RGB: RGB, yang merupakan singkatan dari Red, Green, dan Blue, adalah metode yang digunakan dalam CSS untuk mendeskripsikan warna. RGB bekerja dengan mencampurkan jumlah tertentu dari tiga warna primer, masing-masing dengan nilai antara 0 hingga 255. Dengan menyesuaikan nilai ini, kita dapat menghasilkan berbagai macam warna, memungkinkan terciptanya palet warna yang beragam untuk situs web.

Properti Nilai Warna RGB: Beberapa propertinya adalah:
- Representasi RGB ditulis sebagai rgb(red, green, blue).
- Dengan menyesuaikan nilai dari 0 hingga 255, kita dapat menghasilkan 16.777.216 warna unik.
- Misalnya, rgb(0, 255, 0) menghasilkan warna hijau karena nilai hijau berada di maksimum (255), sementara merah dan biru bernilai 0.
- Sebaliknya, rgb(0, 0, 255) menghasilkan biru, dengan kanal biru diatur ke maksimum (255), dan merah serta hijau diatur ke 0.
- Untuk menghasilkan hitam, semua parameter warna diatur ke 0 (rgb(0, 0, 0)), dan untuk putih, semua parameter diatur ke maksimum (rgb(255, 255, 255)).

// Blue background
<p style="background-color: rgb(0, 0, 255);"> Is the sky background Blue by using RGB </p>

HTML Tag

Apa itu Tag HTML ?

Tag dalam HTML adalah elemen dasar yang digunakan untuk menentukan struktur dan konten pada halaman web. Tag biasanya ditulis dalam pasangan pembuka dan penutup, dengan konten berada di antaranya.


Struktur Dasar Tag HTML:

- Tag Pembuka: Memberitahu browser jenis elemen yang akan digunakan. Ditulis dalam kurung sudut (<>), misalnya: <p>.

- Tag Penutup: Menandai akhir dari elemen, ditulis dengan menambahkan garis miring (/) sebelum nama tag, misalnya: </p>.

- Konten: Teks, gambar, atau elemen lainnya yang berada di antara tag pembuka dan penutup.

Contoh Tag Populer dalam HTML:

Tag Teks:
- <h1> hingga <h6>: Untuk heading (judul).
- <p>: Untuk paragraf.
- <b> atau <strong>: Untuk teks tebal.
- <i> atau <em>: Untuk teks miring.

Tag Media:
- <img>: Untuk gambar.
- <video>: Untuk video.
- <audio>: Untuk audio.

Tag Tabel:
- <table>: Untuk membuat tabel.
- <tr>: Untuk baris tabel.
- <td>: Untuk data tabel.

Tag Hyperlink:
- <a href="url">: Untuk membuat tautan.

HTML Block & Inline

Apa itu HTML Block & Inline?

Dalam HTML, elemen Block dan Inline adalah kategori elemen berdasarkan bagaimana elemen tersebut ditampilkan di halaman web. Keduanya memiliki peran penting dalam menentukan struktur dan tata letak sebuah halaman web.


Block Element

Block element dalam HTML adalah elemen yang secara default tampil sebagai blok atau bagian terpisah di halaman web. Elemen ini akan selalu memulai baris baru dan mengambil seluruh lebar yang tersedia dari kontainer induknya, terlepas dari panjang kontennya.

Karakteristik Utama Block Element:

- Dimulai dari Baris Baru: Setiap kali sebuah elemen blok digunakan, elemen berikutnya akan dimulai dari baris baru.

- Lebar Penuh Kontainer: Elemen blok akan mengambil seluruh lebar kontainer induknya secara otomatis, bahkan jika kontennya pendek.

- Digunakan untuk Struktur Halaman: Elemen blok sering digunakan untuk menyusun tata letak dan struktur utama halaman web, seperti paragraf, heading, dan daftar.

- Dapat Mengandung Elemen Lain: Elemen blok dapat berisi elemen blok lain atau elemen inline.

Inline Element

Inline Element dalam HTML adalah elemen yang ditampilkan di baris yang sama dengan konten atau elemen lainnya. Elemen ini tidak memulai baris baru secara default, dan lebar elemen hanya sebesar kontennya, bukan seluruh lebar kontainer.

Karakteristik Utama Inline Element:

- Tidak Dimulai dari Baris Baru: Inline elements tetap berada dalam baris yang sama dengan teks atau elemen lainnya.

- Lebarnya Hanya Sesuai Konten: Inline elements tidak mengambil seluruh lebar kontainer. Lebarnya hanya mengikuti panjang konten yang ada di dalamnya.

- Digunakan untuk Konten Kecil: Inline elements digunakan untuk elemen kecil yang biasanya menjadi bagian dari elemen blok, seperti teks tebal, tautan, atau gambar.

- Tidak Bisa Berisi Elemen Block: Elemen inline hanya bisa berisi teks atau elemen inline lainnya, tidak dapat berisi elemen blok seperti <div> atau <p>.
