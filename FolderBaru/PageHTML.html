Apa itu HTML?


HTML, yang merupakan singkatan dari HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML adalah inti dari setiap halaman web, memungkinkan pembuatnya untuk mendesain struktur dan konten sebuah situs yang dapat diakses melalui browser.

Dengan Code Editor ini anda dapat mencoba Membuat HTML anda sendiri!

Contoh Sederhana
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Penjelasan

  • <!DOCTYPE html>: Deklarasi ini mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
  • <html>: Elemen ini adalah elemen akar (root) dari sebuah halaman HTML.
  • <head>: Elemen ini berisi informasi meta tentang halaman HTML, seperti judul dan referensi ke file CSS atau JavaScript.
  • <title>: Elemen ini menetapkan judul halaman HTML (judul ini akan ditampilkan di tab browser).
  • <body>: Elemen ini mendefinisikan isi dokumen yang akan terlihat di halaman, seperti heading, paragraf, gambar, tautan, tabel, dan elemen lainnya.
  • <h1>: Elemen ini digunakan untuk mendefinisikan heading utama (paling besar).
  • <p>: Elemen ini digunakan untuk mendefinisikan sebuah paragraf teks.
