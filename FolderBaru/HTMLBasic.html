HTML List

Apa Itu HTML List?


HTML List adalah elemen dalam HTML yang digunakan untuk membuat daftar item. Daftar ini bisa berupa daftar berurutan, daftar tidak berurutan, atau daftar deskripsi. Elemen daftar sangat berguna untuk menyusun informasi dalam bentuk poin atau format terstruktur sehingga lebih mudah dibaca dan dipahami oleh pengguna.


Jenis - Jenis List

1. Unordered List

Unordered List adalah daftar yang tidak memiliki urutan tertentu. Daftar ini digunakan untuk menampilkan item yang tidak perlu diurutkan, misalnya daftar belanja atau daftar poin informasi. Setiap item dalam daftar ini diberi simbol (bullet) secara default, seperti lingkaran hitam kecil, tetapi simbol ini dapat diubah dengan menggunakan atribut atau CSS. Contohnya :

An Unordered HTML list:
  • First item
  • Second item
  • Third item
  • Fourth item

Contoh Codingan Dari HTML Unordered List ini :

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
  <li>Fourth item</li>
</ul>

2. Ordered List

Ordered List adalah daftar yang memiliki urutan atau hierarki tertentu. Daftar ini biasanya digunakan untuk menampilkan langkah-langkah, prioritas, atau urutan proses. Item dalam daftar ini diberi nomor atau huruf secara default.

An Ordered List HTML:
  1. First Item
  2. Second Item
  3. Third Item
  4. Fourth Item

Contoh Codingan Dari HTML Ordered List:

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
  <li>Fourth item</li>
</ol>

HTML Attributes

Apa itu HTML Attributes?

HTML Attributes adalah properti tambahan yang digunakan untuk memberikan informasi lebih lanjut tentang elemen HTML. Atribut ini digunakan untuk mengubah atau menentukan perilaku elemen, seperti menentukan URL untuk tautan, ukuran gambar, atau menambahkan identitas unik pada elemen.


Contoh-Contoh Attributes

Attribut id : Memberikan Identitas Unik pada Tag

<p id="paragraf1">Ini adalah paragraf pertama.</p>

Attribut class: Mengelompokkan Elemen untuk Tujuan Styling

<p class="highlight">Paragraf ini memiliki kelas "highlight".</p>

Attribut Style: Menambahkan Gaya Langsung Ke Elemen

<p style="color: red;">Teks ini berwarna merah.</p>

HTML Table

Apa itu HTML Table?

HTML Table adalah elemen dalam HTML yang digunakan untuk menampilkan data dalam bentuk tabel. Tabel terdiri dari baris dan kolom yang memudahkan penyajian informasi yang terorganisir, seperti data statistik, jadwal, daftar, atau laporan.

Struktur Dasar HTML Table: Sebuah tabel dalam HTML dibuat menggunakan elemen <table>. Tabel dapat memiliki elemen-elemen pendukung seperti:

  • <tr> (Table Row): Mendefinisikan baris dalam tabel.
  • <th> (Table Header): Mendefinisikan sel header dalam tabel, biasanya digunakan untuk memberikan judul kolom.
  • <td> (Table Data): Mendefinisikan sel data biasa dalam tabel.

Contoh Codingan dari Tabel Sebagai Berikut :

<table>
  <thead>
    <tr>
      <th>Nama</th>
      <th>Usia</th>
      <th>Kota</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ali</td>
      <td>25</td>
      <td>Jakarta</td>
    </tr>
    <tr>
      <td>Siti</td>
      <td>30</td>
      <td>Bandung</td>
    </tr>
  </tbody>
</table>

Output dari Codingan diatas adalah sebagai berikut :

Nama Usia Kota
Ali 25 Jakarta
Siti 30 Bandung

Apabila Code diberi sentuhan CSS :

table {
    border-collapse: collapse;
    width: 100%;
}

th, td {
    border: 2px solid black;
    padding: 10px;
    text-align: left;
}

th {
    background-color: #333;
    color: white;
}

tr:nth-child(even) {
    background-color: #f2f2f2;
}

Maka Outputnya akan Menjadi :

Nama Usia Kota
Ali 25 Jakarta
Siti 30 Bandung

Elemen <table> adalah elemen utama untuk membuat tabel di HTML. Semua elemen tabel seperti <thead>, <tbody>, <tr>, <th>, dan <td> berada di dalam elemen <table>.

Elemen <thead> adalah bagian header tabel. Elemen ini digunakan untuk mendefinisikan baris header yang biasanya berisi judul kolom.

Elemen <tr> (Table Row) digunakan untuk menentukan baris dalam tabel. Semua elemen <th> (header) atau <td> (data) akan berada di dalam elemen <tr>.

Elemen <th> (Table Header) digunakan untuk membuat sel header dalam tabel. Dalam kode ini, kolom tabel memiliki header: "Nama", "Usia", dan "Kota". Biasanya, teks dalam elemen <th> akan otomatis dicetak tebal dan rata tengah (tergantung pada gaya default browser).

Elemen <tbody> adalah bagian yang digunakan untuk mendefinisikan konten utama tabel, yaitu data yang ada di bawah header. Semua baris data diletakkan di dalam elemen <tbody>.

Elemen <td> (Table Data) digunakan untuk membuat sel data biasa dalam tabel. Dalam kode ini, elemen <td> digunakan untuk menampilkan data seperti: "Ali", "25", "Jakarta" untuk baris pertama, dan "Siti", "30", "Bandung" untuk baris kedua.
