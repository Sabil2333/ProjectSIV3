Selamat Datang di Tutorial HTML

Apa itu HTML?


HTML, yang merupakan singkatan dari HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML adalah inti dari setiap halaman web, memungkinkan pembuatnya untuk mendesain struktur dan konten sebuah situs yang dapat diakses melalui browser.

Dengan Code Editor ini anda dapat mencoba Membuat HTML anda sendiri!

Contoh Sederhana
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
Try it Yourself »

Penjelasan

  • <!DOCTYPE html>: Deklarasi ini mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
  • <html>: Elemen ini adalah elemen akar (root) dari sebuah halaman HTML.
  • <head>: Elemen ini berisi informasi meta tentang halaman HTML, seperti judul dan referensi ke file CSS atau JavaScript.
  • <title>: Elemen ini menetapkan judul halaman HTML (judul ini akan ditampilkan di tab browser).
  • <body>: Elemen ini mendefinisikan isi dokumen yang akan terlihat di halaman, seperti heading, paragraf, gambar, tautan, tabel, dan elemen lainnya.
  • <h1>: Elemen ini digunakan untuk mendefinisikan heading utama (paling besar).
  • <p>: Elemen ini digunakan untuk mendefinisikan sebuah paragraf teks.

HTML Links

Apa itu Link dalam HTML?


HTML Link adalah elemen dalam HTML yang digunakan untuk membuat tautan (hyperlink) yang menghubungkan satu halaman web dengan halaman lain, baik di dalam situs yang sama maupun ke situs lain di internet. Elemen ini memungkinkan pengguna untuk berpindah dari satu dokumen, sumber daya, atau lokasi tertentu ke lokasi lainnya dengan mudah.

HTML Link - Syntax

<a href="url">Link text</a>

Tag <a> digunakan untuk membuat hyperlink. Atribut href menentukan URL tujuan, seperti "url".


Berikut Merupakan Contoh dari Codenya

<a href="https://SabilCourse.com/">Visit W3Schools.com!</a>

Tag <a> digunakan untuk membuat hyperlink. Atribut href menentukan URL tujuan, seperti "SabilCoure.www". Kemudian Link akan mengarahkan anda ke Website atau URL yang dituju.

HTML Image

Apa Itu HTML Image?


HTML Image adalah elemen dalam HTML yang digunakan untuk menampilkan gambar di halaman web. Elemen ini memungkinkan pengembang web untuk menambahkan berbagai jenis gambar, seperti foto, ilustrasi, grafik, atau logo, sehingga membuat halaman web menjadi lebih menarik dan informatif.

HTML Image didefinisikan menggunakan elemen <img> (image tag), yang merupakan elemen self-closing (tidak memerlukan tag penutup).


Contoh dari Image di HTML


Gambar diatas Merupakan contoh dari Penggunaan dan pengaturan <image>


HTML Image - Syntax

Contoh Syntax Image
<img src="Fotokucing.webp"  alt="FotoKucing" >
<img src="Pemandangan.jpg"  alt="Pemandangan">
<img src="Fotokucing.webp"  alt="FotoKucing" width="200" height="200">
Try it Yourself »

Kode di atas menggunakan elemen <img>, yang merupakan elemen self-closing dalam HTML. Elemen ini digunakan untuk menampilkan gambar pada halaman web. Pada atribut src, ditentukan sumber gambar yang ingin ditampilkan, seperti Fotokucing.webp atau Pemandangan.jpg.

Atribut alt memberikan deskripsi alternatif untuk gambar jika gambar gagal dimuat. Selain itu, atribut width dan height digunakan untuk menentukan ukuran gambar dalam piksel, contohnya 200 x 200.
