Komentar dalam C++


Apa itu Komentar dalam C++?

Komentar dalam C++ digunakan untuk memberikan penjelasan pada kode dan meningkatkan keterbacaan. Komentar juga dapat digunakan untuk menonaktifkan kode tertentu saat pengujian.


Jenis-Jenis Komentar

Dalam C++, terdapat dua jenis komentar:


Komentar Satu Baris

Komentar satu baris dimulai dengan //. Semua teks setelah // hingga akhir baris akan diabaikan oleh kompiler.

Contoh Komentar Satu Baris
#include <iostream>
using namespace std;

int main() {
    // Ini adalah komentar satu baris
    cout << "Hello, Dunia!";
    return 0;
}

Komentar Multi-Baris

Komentar multi-baris dimulai dengan /* dan diakhiri dengan */. Semua teks di antara /* dan */ akan diabaikan oleh kompiler. 

#include <iostream>
using namespace std;

int main() {
    /*
    Ini adalah komentar
    multi-baris
    */
    cout << "Hello, Dunia!";
    return 0;
}

Penggunaan Komentar

  • Gunakan komentar satu baris untuk penjelasan singkat.
  • Gunakan komentar multi-baris untuk menjelaskan blok kode yang lebih kompleks.
  • Hindari penggunaan komentar yang terlalu berlebihan sehingga mengurangi keterbacaan kode.
